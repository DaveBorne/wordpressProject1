-- BEFORE --
Install node.js - https://nodejs.org/en/download/


1)	Copy WP files to directory
2)	Run install, setting up DB connection and Site Information
3)	Site now using bornePress theme - This automatically changes

4)	Install gulp dependancies - type cd then drag theme folder to terminal then press enter
5)	In terminal type npm install --only=dev then press enter
6)	After the install has finished - type gulp then press enter

4)	Run gulp workflow
5)	Files in app folder you can edit, try to keep structure as is.
6)	Don't edit files in dist folder directly - these are generated from the workflow
